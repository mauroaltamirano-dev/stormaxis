# User Context — tuki

Este archivo es una base viva para personalizar el trabajo entre chats. No debe guardar secretos, tokens, contraseñas, datos bancarios ni información sensible. Si algo es privado o dudoso, se resume de forma segura o se deja fuera.

## Cómo actualizarlo

- Agregar datos sólo cuando ayuden a trabajar mejor juntos.
- Preferir preferencias, forma de trabajo, decisiones recurrentes y contexto de producto.
- Si el dato es personal sensible, pedir confirmación explícita antes de escribirlo.
- Mantenerlo corto y fácil de leer.
